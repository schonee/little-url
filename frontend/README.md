# 🎨 Frontend - Little URL

This is the frontend for the URL shortener project, built with React, Vite, TypeScript, and Tailwind CSS.

## 🌟 Features

- Input for original URL
- Displays shortened URL
- Tailwind styling
- Fetches from backend API

## 🧪 Requirements

- Node.js 18+
- Backend running

## 📦 Setup

```bash
cd frontend
npm install
```
